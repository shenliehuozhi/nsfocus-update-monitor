## 二、字段说明

### 2.1 订阅条件 pills 区域

**数据结构：** `chain[]` + `match`（leaf / subtree）

| 元素 | 说明 |
|------|------|
| pill 标签 | 显示完整路径，如 `NFS堡盒 › V6.0.8 › 规则ID分组` |
| 删除按钮 | pill 右侧 ×，点击移除该路径 |
| + 添加路径 | 打开树形路径选择弹窗 |
| 订阅全部复选框 | 勾选后 pills 区域禁用，添加路径按钮变灰 |

**互斥逻辑：**
- 勾选"订阅全部" → 订阅条件 pills 区域 disabled
- 添加了任意 path → "订阅全部"自动取消勾选并 disabled

**路径匹配语义：**
- `leaf`（仅当前节点）：精确到最末级节点
- `subtree`（包含子节点）：包含当前节点及所有子节点

### 2.2 推送模式 tooltip

| 选项 | 说明 |
|------|------|
| 即时推送 | 检测到版本变化立即推送 |
| 定时推送 | 按设定周期推送 |
| 汇总推送 | 按设定时间汇总推送 |

### 2.3 通知渠道

**UI 形式：** 横向标签组 + 添加按钮

```
通知渠道
┌─────────────────────────────────────────────────────────┐
│ [163邮箱 (email)  ✕]  [企业微信  ✕]  [+ 添加渠道]    │
└─────────────────────────────────────────────────────────┘
```

**添加渠道弹窗：** 下拉选择已有渠道（邮件、企微、钉钉等）

**删除：** 标签右侧 × 按钮移除

### 2.4 动态显隐字段

| 字段 | 显示条件 |
|------|---------|
| 客户邮箱 | 通知渠道包含任意 **邮件类型** |
| 附件大小限制 (MB) | 通知渠道包含任意 **邮件类型** |

两个字段同时显示/隐藏，条件一致。

## 四、路径选择弹窗（已有基础，待优化）

### 4.1 交互流程

1. 点击 "+ 添加路径" → 打开 `openChainSelectorModal`
2. 左侧树：可折叠展开，叶子节点可单独选中
3. 中间节点：选中自动为 subtree（包含子节点）
4. 右侧实时列表：显示已选路径 + match 类型标签（leaf / subtree）
5. 点确定 → 数据写入 `window._condChains` → 关闭弹窗
6. 创建规则页面 pills 区域更新显示

### 4.2 冲突检测

父子路径互斥：
- 选了父节点后，子节点不可选（闪红提示）
- 选了子节点后，父节点不可选（闪红提示）
